# NotesApp

Hi There

This is my jetpack compose note app.


So far I have 4 composable screens, which are:
1. NotesScreen
2. DetailsNoteScreen
3. NewNoteScreen
4. EditNoteScreen

This is being done based on MVVM and clean arquiteture.
